## Опис проєкту

**Scanlate Project Tracker** це вебзастосунок для організації роботи над перекладацькими та редакторськими проєктами. Система дозволяє користувачу створювати команди, додавати проєкти, вести список поточних задач, відстежувати дедлайни, переглядати статистику оплати та переносити завершені задачі в архів.

Інтерфейс побудований у вигляді окремих сторінок:

- `login.html` і `register.html` для входу та реєстрації;
- `main.html` для перегляду поточних задач і статистики;
- `calendar.html` для візуалізації дедлайнів;
- `archive.html` для перегляду завершених задач та експорту;
- `add.html` для створення команд і проєктів;
- `profile.html` для редагування профілю користувача.

---

## Основний функціонал

- реєстрація користувача через email і пароль;
- вхід через email/пароль або Google-акаунт;
- створення команд із логотипами;
- створення проєктів із зображеннями обкладинок;
- додавання задач із типом, номером глави, дедлайном і ціною;
- перегляд активних задач у табличному вигляді;
- пошук по задачах;
- перегляд статистики: зароблено, виплачено, до виплати;
- календар дедлайнів з позначенням готових задач;
- архів завершених задач;
- експорт архіву у формат `.xlsx`;
- редагування профілю та аватара;
- збереження даних у Firebase.

## Реалізація даних

У проєкті використовується Firebase як хмарна платформа для збереження й обробки даних:

- `Firebase Authentication` для реєстрації та входу користувачів;
- `Cloud Firestore` для збереження профілів, команд, проєктів, активних і архівних задач;
- `Firebase Storage` для завантаження аватарів, логотипів і зображень проєктів;
- `Firebase Hosting` для публікації вебзастосунку.

Згідно з поточною структурою, дані зберігаються в підколекціях конкретного користувача, що ізолює записи та спрощує персональне використання застосунку.
